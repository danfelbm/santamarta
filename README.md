# Santa Marta & Beyond

Itinerario de viaje para el plan a Santa Marta y el Parque Tayrona (21 al 24 de junio).

Sitio estático de una sola página construido con HTML y [Tailwind CSS](https://tailwindcss.com/).

## Desarrollo

Abre `index.html` directamente en el navegador, o sírvelo con cualquier servidor estático:

```bash
npx serve .
```

## Despliegue

Desplegado como sitio estático en [Vercel](https://vercel.com/). No requiere build: Vercel sirve `index.html` directamente desde la raíz.
